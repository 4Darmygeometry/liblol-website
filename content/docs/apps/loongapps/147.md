{{< issue 默认启动脚本不可用 >}}

单位转换器的默认启动脚本：

```
/usr/local/UnitConverter/start.sh
```

会使用位于如下目录的运行时库：

```
/usr/local/UnitConverter/platforms
```

这会导致各类运行时库不兼容的情况。实际上，在安装相关依赖后，可以使用如下命令启动：

```
/usr/local/UnitConverter/UnitConverter
```

{{< /issue >}}
