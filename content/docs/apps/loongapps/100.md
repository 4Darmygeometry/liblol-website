{{< issue 沙箱功能导致浏览器渲染引擎崩溃 >}}
该浏览器默认开启沙箱功能，浏览器启动后无法渲染任何网页，报错：

```
喔唷，崩溃啦！
显示此网页时出了点问题。

错误代码：SIGSEGV
```

且一段时间后，整个浏览器界面都会失去响应。该问题是浏览器使用的 Chromium 核心过老，沙箱功能不支持给 `statx()` 返回 `NOSYS` 导致的；此时，`statx()` 无法正确回落到 libLoL 兼容的 `fstat()` 系统调用，因而导致问题。

请使用如下命令启动程序：

```bash
qaxbrowser-safe --no-sandbox
```
{{< /issue >}}
